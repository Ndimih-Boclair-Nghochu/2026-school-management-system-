# EduIgnite Frontend

This directory contains the React frontend for the EduIgnite school management system.

## Setup

1. `cd frontend`
2. `npm install` (will install React, react-icons, etc.)
3. `npm start` to launch the development server.

The application currently renders a static **Teacher Dashboard** matching the provided design.

## Structure

- `src/components` – reusable UI components (Sidebar, Header, TeacherDashboard, etc.)
- `src/index.js` / `src/App.js` – entry point wiring up the dashboard

Styles are plain CSS under each component.

Further modules and routing can be added as development continues.
